* while True 문을 사용해 프로그램이 종료(번호 입력) 전까지 무한 반복되게 만들어 프로그램이 처음 켜지면 메인 메뉴(1~5번)를 띄워줍니다. 5번 운동 루틴을 조회해서 오늘 해야할 운동을 확인합니다. 이기능은 리스트 조작을 통해 언제든지 운동루틴을 유동적으로 변경할 수 있습니다. 1번 식단 기록에스는 오늘 먹은 음식의 칼로리와 몸무게를 입력해 하루 섭취 권장량에 맞춰서 먹었는 지 체크합니다 2번 메뉴는 운동 할때 종목별 횟수 와 무게를여러가지 변수와 자료형으로 입력받아  총 볼륨을 산출합니다. 3번 메뉴는 본인의 아침 몸무게를 입력해 BMI수치를 계산하여 현재 본인의 체중상태를 알려줍니다. 4번 메뉴는 체계적인 삶을 살기 위해 자신이 정해둔 루틴 예를 즐어 운동, 전공공부등을 했는 지 기록합니다. 마지막으로 0번을 누르면 프로그램이 종료됩니다


## 3. 예상 기능 및 메뉴 (최소 5개)
1. 식단 기록

4. 오늘의 루틴 체크
5. 운동 루틴 확인
0. 종료

### **🤖 AI 파트너십 과정**
(예시)
 1. **내용 1 : 길드 매니저 컨셉 도출**
    * **프롬프트 요약:** 이번 학기에 파이썬 프로그래밍을 이용해서 4단계(1.변수/입출력➔2.조건문➔3.반복문/메뉴➔4.함수/배열)로 발전 시킬 콘솔프로그램을만들어야해. 컨셉 몇개 만 추천 해줘
    * **적용 내용:** 제미나이가 추천해준 컨셉 중 평소에 운동을 조금씩 하는데 무게나 횟수를 귀찮아서 잘 기록하지 않아 이전 보다 더 성장여부를 알고 싶고 체계적인 삶을 구축하고 싶어서 이 컨셉을 선택했습니다 
      
 2. **내용 2 : 로드맵 도출**
    * **프롬프트 요약:** "1차 과제 필수 요건(변수 5개, 자료형 3개 이상)을 충족해서 운동기록 메뉴를 만들어줘"
    * **적용 내용:**  운동 총 볼륨을 기록할 때 필요한 요소들을 문자형(`char` -운동 종목), 정수형(`int` - 세트수, 반복횟수), 실수형(`float` - 운동 무게)으로 자료형을 명확히 분리하여 입력 데이터의 구조적 완성도를 높입니다
